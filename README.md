BeliTrophy adalah aplikasi web katalog trophy yang menampilkan gambar, kategori, dan harga produk, memungkinkan pengguna memilih beberapa trophy sekaligus, menentukan jumlah, dan mengirim pesanan langsung melalui WhatsApp tanpa login. Dibangun menggunakan Next.js dengan tampilan berbasis HTML dan CSS, aplikasi ini memanfaatkan Prisma.io untuk pengelolaan data produk dan Apidog untuk dokumentasi serta pengujian API. Selain halaman user, BeliTrophy memiliki dashboard admin pada URL khusus yang dilindungi login, di mana admin dapat melakukan CRUD produk serta memantau dan memperbarui status pesanan, sehingga peran antara user dan admin tetap terpisah dan aman dalam satu aplikasi.
